# SafePay BNPL Application

A full-stack Buy-Now-Pay-Later platform with role-based dashboards.

## Quick Start

```bash
npm install
npm run dev
```

The app will open in your browser at http://localhost:5173

## Features

- **Merchant Portal**: Initiate BNPL checkouts and track transactions
- **Employer Portal**: Manage employee deductions and payroll
- **Admin Dashboard**: Monitor contracts and reconciliation
- **Customer View**: Check BNPL limits and payment schedules

## Default Demo Credentials

- Use **any username** when logging in
- Select your role: Merchant, Employer, Admin, or Customer
- All data is demo/mock data for testing

## Building

```bash
npm run build
npm run preview
```
